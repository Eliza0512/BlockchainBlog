# Azure Blockchain Service是什么？

根据官方文档，微软的服务主要提供

- 简单的操作帮助用户简历区块链网络和运维
- 内置的共识管理系统
- 提供常用工具便于智能合约的开发

目前AzureBlockchianService(一下简称ABS)提供的区块链模型是企业级以太坊账本Quorum（关于Quorum的详细介绍可参见博主博客），共识采取IBFT（后续分析中）。
